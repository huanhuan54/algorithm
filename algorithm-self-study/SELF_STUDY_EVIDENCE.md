# 算法自学项目说明

## 项目定位

本仓库是一个算法与数据结构自学项目，重点记录学习过程、专题文章、代码实现、测试用例和阶段复盘。内容不是一次性整理的结论，而是按专题逐步补充，方便回看自己的理解变化。

## 内容主题

自学主题：算法与数据结构基础

覆盖内容：

- 算法学习路线与复杂度分析
- 数组、双指针与滑动窗口
- 栈、队列与单调栈
- 哈希表
- 递归、搜索与回溯
- 动态规划

## 文章位置

文章统一存放在 `articles/` 目录：

## 配套代码

配套算法代码存放在 `src/` 目录，测试代码存放在 `tests/` 目录。可通过以下命令验证：

```bash
python -m unittest discover -s tests
```
