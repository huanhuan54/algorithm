# 练习题型索引

这份索引用来记录每个专题对应的典型题型，方便后续继续补题。

## 复杂度与二分

典型题型：

- 在有序数组中查找目标值。
- 查找插入位置。
- 查找重复元素的左右边界。

当前实现：

- `binary_search`
- `lower_bound`
- `upper_bound`
- `search_insert_position`

## 数组与双指针

典型题型：

- 有序数组两数之和。
- 固定长度窗口最大和。
- 原地移动或删除元素。

当前实现：

- `two_sum_sorted`
- `max_sum_subarray`

## 哈希表

典型题型：

- 普通两数之和。
- 统计字符或数字频次。
- 判断是否出现重复元素。

当前实现：

- `two_sum`
- `first_unique_char`

## 前缀和

典型题型：

- 多次区间和查询。
- 统计和为目标值的连续子数组数量。
- 包含负数的连续子数组问题。

当前实现：

- `build_prefix_sum`
- `range_sum`
- `subarray_sum_count`

## 栈与队列

典型题型：

- 括号匹配。
- 单调栈求下一个更大元素。
- BFS 层序扩展。

当前实现：

- `is_valid_parentheses`
- `next_greater_elements`

## 回溯

典型题型：

- 子集。
- 排列。
- 组合。
- 棋盘类搜索。

当前实现：

- `subsets`
- `permutations`

## 动态规划

典型题型：

- 爬楼梯。
- 股票买卖。
- 最长递增子序列。

当前实现：

- `climb_stairs`
- `max_profit`
- `length_of_lis`

## 树与图

典型题型：

- 二叉树前中后序遍历。
- 二叉树层序遍历。
- 图的 DFS/BFS。
- 无权图最短路径。

当前实现：

- `preorder`
- `inorder`
- `postorder`
- `level_order`
- `dfs_order`
- `bfs_order`
- `shortest_path_length`
