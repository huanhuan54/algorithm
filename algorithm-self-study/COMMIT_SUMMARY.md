# 学习提交记录

本仓库保留了按学习专题推进的提交记录。每次提交尽量只处理一个明确的学习目标，例如新增一个专题、补充测试、完善复盘或整理学习索引。

| 提交内容 | 学习内容 |
| --- | --- |
| `Add files via upload` | 初始上传算法自学文章、代码和测试。 |
| `Clean evidence repository contents` | 清理缓存文件，保留源码、文章和测试。 |
| `Add repository evidence overview` | 增加仓库首页说明，便于审核定位材料。 |
| `Add learning log and review notes` | 增加学习日志和错题复盘，记录真实学习过程。 |
| `Add prefix sum study topic` | 增加前缀和专题文章、代码和测试。 |
| `Expand binary search boundary notes` | 扩展二分查找边界问题，增加 `lower_bound` 和 `upper_bound`。 |
| `Add sorting basics study topic` | 增加排序专题文章、选择排序、插入排序、归并排序和测试。 |
| `Add binary tree traversal notes` | 增加二叉树遍历专题、树节点结构和遍历测试。 |
| `Add graph search study topic` | 增加图搜索专题、DFS、BFS、无权最短路径和测试。 |
| `Add practice index for study topics` | 增加练习题型索引，把文章、代码和题型对应起来。 |
| `Implement binary answer study topic` | 实现二分答案专题，补充运载能力和吃香蕉速度两个练习。 |
| `Implement greedy algorithm study topic` | 实现贪心专题，补充区间调度、跳跃游戏和股票多次交易。 |
| `Implement union find study topic` | 实现并查集专题，补充动态连通关系和连通块数量测试。 |

## 复盘

这些提交对应文章、算法实现、测试用例、学习日志、复盘记录和项目说明。通过提交历史可以看到学习内容从基础查找逐步扩展到前缀和、排序、树、图搜索、二分答案、贪心和并查集。
